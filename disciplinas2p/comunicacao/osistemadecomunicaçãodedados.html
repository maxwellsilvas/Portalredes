◉ Equipamentos de comunicação de dados

Os computadores de uma empresa precisam estar interligados para facilitar a comunicação. Para isso, é necessário construir uma estrutura formada por equipamentos de rede específicos.

Muitas atividades dependem dessa conexão, como o compartilhamento e armazenamento de arquivos, o funcionamento de serviços de softwares, além da comunicação interna e externa.

Embora esse seja um assunto bastante técnico, ter certo conhecimento a respeito é importante para fazer boas escolhas ao contratar esse serviço para a sua empresa. Que tal começar conhecendo melhor os componentes utilizados?

Nesta postagem iremos listar os principais equipamentos de rede e a sua função na infraestrutura. Vamos começar a desvendá-los?

O que são equipamentos de rede?
Hoje, uma empresa não consegue operar sem o uso da internet e de sistemas para gerenciar os seus processos. Mas, para muitos dos chamados softwares funcionarem, é preciso ter uma estrutura física, conhecida como hardware.

Os equipamentos de rede são hardwares, dispositivos físicos necessários para a comunicação entre os computadores de uma organização. Após devidamente conectados, eles formam uma infraestrutura de rede.

A forma como uma rede será criada e, consequentemente, os equipamentos utilizados, dependem das necessidades de cada negócio, como o número de computadores para integrar, os recursos a serem compartilhados, o tipo de segurança adotada, etc. Em um supermercado, por exemplo, os computadores do estoque e dos caixas precisam se comunicar para registrar e dar baixa nas mercadorias que foram vendidas. Esses dados são transmitidos através da rede interna, também chamada de intranet, que depende de diversos equipamentos para funcionar.

Quais são os equipamentos de uma rede?
Por mais que pareça um assunto complexo à primeira vista, com um pouco de paciência e leitura é possível compreender a função de cada equipamento da rede, acompanhe!

➤ Cabos
Os cabos são equipamentos básicos de uma rede. Eles são os responsáveis por realizar as ligações para formar a rede e também transmitir o sinal de internet. Podem ser encontrados em diferentes tipos, sendo os mais usados os de fibra óptica e de par trançado.

Caixa de emenda óptica
As empresas que optarem por usar cabos de fibra óptica, poderão precisar dessa caixa para unir dois pontos em uma emenda. Por ser um material delicado, essas ligações devem ser feitas somente com esse dispositivo.

➤ Firewall
O firewall é um equipamento voltado para a proteção da rede, formando uma barreira de segurança.

Quando colocado entre a rede local e a conexão com a internet, evita a invasão de agentes maliciosos. Através dele, a equipe de TI conseguirá controlar acessos a partir dos computadores da empresa.

➤ Gateway
Cada rede tem um protocolo, um conjunto de regras para a comunicação entre dois ou mais dispositivos. Em alguns casos, redes com protocolos diferentes precisam ser ligadas, e o gateway realiza esse intercâmbio, evitando o bloqueio dos dados.

➤ OLT
OLT é a sigla para Optical Line Terminal, em português, terminal de linha óptica. Esse dispositivo é responsável por conectar equipamentos de fibra óptica e transferir os sinais.

➤ ONU
ONU é mais uma sigla relacionada à fibra óptica e significa Optical Network Unit, ou Unidade de Rede Óptica. Esse equipamento de rede funciona como um terminal em uma rede de fibra óptica, fornecendo aos usuários múltiplas interfaces do serviço.

➤ Roteador
O roteador é um equipamento presente na maioria das residências que possuem conexão com a internet, conectando duas ou mais redes. Seu uso mais comum é para conectar uma rede local à internet, o que pode ser feito com ou sem fio, no caso de dispositivos com Wi-Fi.

➤ Repetidor
Os repetidores são complementos dos roteadores, utilizados para expandir o alcance de um sinal de conexão sem fio. Se uma empresa tem uma área muito grande, que não dá para ser coberta pelo sinal de internet apenas com roteadores, são usados repetidores.

➤ Splitter
A tradução da palavra splitter é divisor, sendo exatamente essa a função desse equipamento de rede. É aplicado em redes de fibra óptica para dividir o sinal.

Existem dois tipos de splitter, o balanceado e o não-balanceado. O primeiro faz a divisão igualmente, gerando perda no sinal, enquanto o segundo mantém a qualidade do cabo original.

Curiosidade: o uso de splitter é bastante comum em regiões que estão começando a implantar a internet de fibra óptica.

➤ Bridge
Conhecido como bridge ou ponte, esse equipamento conecta duas redes LAN (Local Area Networks). Essa sigla se refere às redes locais, ou seja, formadas por computadores presentes em um mesmo espaço físico.

Se em uma empresa houver mais de uma rede, esse componente é capaz de uni-las, servindo como uma ponte. Com ele, é possível fazer ligações específicas, apenas para uma ou mais máquinas.

➤ Hub
O hub é um dos primeiros equipamentos que surgiram para a criação de redes de computadores. Por isso, não tem um funcionamento otimizado como opções mais modernas, sobrecarregando a transferência de dados. Hoje, não é muito usado em empresas, sendo mais indicado para redes domésticas ou pequenos negócios.

➤ Switch
O switch é semelhante ao hub e ao bridge, no entanto, é um equipamento mais poderoso. Ele armazena os registros dos dispositivos da rede e transfere pacotes de dados apenas para os que estiverem selecionados.

Sua função é reduzir a latência (atraso na comunicação) de uma rede quando a demanda é alta, pois os dados são transmitidos de maneira otimizada.
